# Display Item Plan

Create or update this file before plotting new main-text figures or tables. The
goal is to make the Researcher's display choices claim-led and auditable without
turning them into a fixed framework recipe.

This plan is separate from `display_item_explanations.md`. The plan records why
the Researcher expects each display item to be necessary; the explanation file
records what the finished item shows and how readers should interpret it.

Shared style helpers are allowed. Shared figure programs are not. Using
`agents/templates/figure_style.py` for fonts, label humanization, line widths,
marker sizes, grids, and saving is acceptable; using a framework-default list
such as ROC curves, calibration, subgroup metrics, and coefficient profile as an
automatic figure sequence is not.

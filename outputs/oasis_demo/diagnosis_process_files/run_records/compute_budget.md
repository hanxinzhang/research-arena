# Compute Budget

target_cpu_core_hours_per_researcher: 0.25
minimum_cpu_core_hours_per_researcher: 0.0001
minimum_experiment_rows_per_researcher: 3

Backend: local Python in the `ag` conda environment.

Minimum work:

- Phase 0 pilot;
- revision_00 analysis;
- revision_01 issue-linked update;
- PDF rendering and audit checks.

The compact demo budget is intentionally small. The target is a ceiling
and planning reference, not a requirement to consume compute when the
approved analyses are already complete. Under-target runs require a
structured compute reconciliation and an Editor-approved efficiency
override before acceptance.

# Dataset Profile

The local OASIS table has 436 rows and 13 columns. The Clinical Dementia
Rating label is observed for 235 rows and missing for 201 rows. Observed
label counts are 135 rows with CDR 0, 70 with CDR 0.5, 28 with CDR 1.0,
and 2 with CDR 2.0.

Predictors considered in the demo include age, sex, education,
socioeconomic status, Mini-Mental State Examination, estimated total
intracranial volume, normalized whole-brain volume, and atlas scaling
factor. Row names and subject identifiers are not predictors.
